PARIVESH 2.0

Description: PARIVESH is a web-based, role-based workflow application designed for online submission and monitoring of proposals seeking Environment, Forest, Wildlife, and CRZ Clearances from various authorities.

Technologies Used: Microservices architecture, Core Java, PostgreSQL, Spring Cloud, Spring Boot, Hibernate, Kafka, Docker, Jenkins, Flowable process engine

Achievements: Developed a web-based workflow application facilitating online submission and monitoring of proposals. Implemented a custom Flowable process engine to enhance workflow management capabilities.
